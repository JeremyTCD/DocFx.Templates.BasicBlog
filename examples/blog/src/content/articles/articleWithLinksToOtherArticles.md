# Linking Using UIDs
## Inferred Link Text
`<xref:about>` is rendered as <xref:about>.
## Explicit Link Text
`[About](xref:about)` is rendered as [About](xref:about).

# Linking Using URLs
`[About](/about.html)` is rendered as [About](/about.html).
